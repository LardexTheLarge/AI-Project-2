## Introduction

## Introduction

In Project 2, our group developed a generic machine learning pipeline capable of addressing a wide range of real-world problems across various industries. We designed a flexible framework that allows users to input their data and obtain meaningful results through data analysis, visualization, and model building. Leveraging the skills acquired in our coursework, we created a versatile solution using Python's Scikit-learn library and other essential technologies. This pipeline empowers users to tackle diverse challenges by simply plugging in their datasets, making it a valuable tool for data-driven decision-making and problem-solving.

## Features

- Washington Real Estate Data: Utilized 2014 Washington real estate data as the primary dataset for analysis and model training.
- Generic Pipeline: Developed a versatile machine learning pipeline capable of preprocessing and modeling any dataset, allowing for seamless integration of diverse data sources.
- Flexible Framework: Designed a modular and adaptable framework that accommodates different data formats and characteristics, enabling users to apply the pipeline to various real-world problems.
- Data-Driven Decision Making: Empowered users to make informed decisions based on data insights derived from the analysis and modeling process, facilitating effective problem-solving and strategic planning.
- Scalable Solution: Engineered a scalable solution that can handle large datasets efficiently, ensuring reliable performance across different data sizes and complexities.

## Getting Started

### Prerequisites

- Python 3.x
- Jupyter Notebook
- Scikit-learn
- Matplotlib
- Pandas

### Installation

1. Clone the repository to your local machine:

```bash
git clone git@github.com:LardexTheLarge/AI-Project-2.git
```

2. Navigate to the project directory:

```bash
cd AI-Project-2
```

3. Run scripts inside each notebook.

### Usage

1. Open the provided notebooks.
2. Run each notebook with the "run all" button.
